
# Lecture Notes - Fonts and Colors (Video Lecture 19)

**index.html (Fonts):**
- **DOCTYPE Declaration:** The document begins with a standard HTML5 declaration.
- **HTML Structure:** The HTML structure includes `<html>`, `<head>`, and `<body>` tags.
- **Character Encoding:** A `meta` tag specifies the character encoding as UTF-8.
- **Title:** The `<title>` tag defines the title of the page as "Fonts."

**CSS Styling for Fonts:**
- The `<style>` block within the `<head>` section contains CSS rules for font styles:
  - The `@import` rule imports font styles from Google Fonts, including "Baloo Bhai 2" and "Poppins."
  - A rule for the `<h1>` element sets the font family to 'Poppins' and includes backup font options.
  - A rule for the `<p>` element sets the font family to 'Baloo Bhai 2', adjusts the font size to 20px, and comments out some font style properties.
  - The `<h2>` element is styled with text alignment, text transformation (uppercase), and text decoration properties.
  - A class named "lorem" is defined, applying a red border, a specific width, and a word-break property for text inside it.

**HTML Elements:**
- The `<body>` section contains a `<div>` element that includes various text elements.
- The `<h1>` element serves as the main heading for the page.
- The `<h2>` element is used to introduce the topic of fonts.
- Two `<p>` elements provide textual content. One of them has a class "lorem" for additional styling.
